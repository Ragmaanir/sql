    class Atom::Repository

    • Atom::Repository
    • Reference
    • Object

    Overview

    A gateway between models and DB. Its main features are logging, expanding Atom::Query instances and mapping models from resulting DB::ResultSet.

    repo = Atom::Repository.new(DB.open(ENV["DATABASE_URL"]), Atom::Atom::Repository::Logger::IO.new(STDOUT))

repo.scalar("SELECT 1").as(Int32)
# [postgresql] SELECT 1
# 593μs

repo.scalar("SELECT ?::int", 1).as(Int32)
# ditto

repo.query("SELECT * FROM users")       # Returns raw `DB::ResultSet`
repo.query(User, "SELECT * FROM users") # Returns `Array(User)`
repo.query(User.all)                    # Returns `Array(User)` as well
# [postgresql] SELECT users.* FROM users
# 442μs
# [map] User
# 101μs
